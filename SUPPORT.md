# Support

## How to Get Help

This project uses GitHub Issues to track bugs and feature requests.

### Before Filing an Issue

1. Search existing issues to avoid duplicates
2. Check the [README.md](README.md) for documentation
3. Review the [examples](examples/) directory for usage patterns

### Filing a New Issue

For new issues, please:
- Use a clear and descriptive title
- Provide detailed steps to reproduce (for bugs)
- Include your environment details (Node.js version, OS, etc.)
- Include code samples when relevant

### Questions and Discussions

For general questions about using the library, please use GitHub Discussions if enabled, or file an issue with the "question" label.

## Project Scope

`libvrf-js` is a JavaScript/TypeScript port of [Microsoft's libvrf](https://github.com/Microsoft/libvrf). 

- **For issues with this JavaScript port**: File issues in this repository
- **For issues with the original C++ library**: File issues at [microsoft/libvrf](https://github.com/Microsoft/libvrf)

## Support Policy

Support for this project is community-based and limited to:
- Bug reports via GitHub Issues
- Feature requests via GitHub Issues  
- Documentation improvements via Pull Requests

There is no guaranteed response time or SLA for support requests.
